# Sequence Diagrams

This directory covers the interaction flow for the required API calls.

## Files

### 1. User Registration
- **File:** `0-User_Registration.md`
- **Flow:** Handles new user sign-up validation and persistence.

### 2. Place Creation
- **File:** `1-Place_Creation.md`
- **Flow:** Validates and saves new listings associated with an owner.

### 3. Review Submission
- **File:** `2-Review_Submission.md`
- **Flow:** Processes rating and comment submission for a specific place.

### 4. Fetching Places
- **File:** `3-Fetching_a_List_of_Places.md`
- **Flow:** Retrieves the list of available places from the storage.
